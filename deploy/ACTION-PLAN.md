# SEO Action Plan - Itaara Reserve

## Antes de subir na Hostinger

1. Usar o ZIP atualizado do pacote final.
2. Subir todos os arquivos diretamente dentro de `public_html`.
3. Confirmar que existem na raiz:
   - `index.html`
   - `.htaccess`
   - `robots.txt`
   - `llms.txt`
   - `sitemap.xml`
   - `assets/`
   - `termos-e-condicoes/`
   - `politica-de-privacidade/`
   - `politica-de-cookies-br/`

## No dia da publicação

1. Fazer backup completo do site antigo.
2. Publicar o novo pacote.
3. Testar:
   - `https://itaarareserve.com.br/`
   - `https://itaarareserve.com.br/robots.txt`
   - `https://itaarareserve.com.br/llms.txt`
   - `https://itaarareserve.com.br/sitemap.xml`
   - `https://itaarareserve.com.br/termos-e-condicoes/`
   - `https://itaarareserve.com.br/politica-de-privacidade/`
   - `https://itaarareserve.com.br/politica-de-cookies-br/`
4. Conferir que `http://` redireciona para `https://`.
5. Conferir se `www.itaarareserve.com.br` redireciona para `itaarareserve.com.br`.

## Depois da publicação

1. Rodar PageSpeed Insights mobile e desktop.
2. Adicionar a propriedade no Google Search Console.
3. Enviar o sitemap `https://itaarareserve.com.br/sitemap.xml`.
4. Solicitar inspeção/indexação da home.
5. Testar compartilhamento no WhatsApp/Facebook/LinkedIn para validar Open Graph.
6. Validar juridicamente as páginas de Termos, Privacidade e Cookies.

## Melhorias opcionais

1. Criar uma versão menor do mapa para mobile.
2. Criar uma imagem Open Graph 1200x630 exclusiva para compartilhamento.
3. Adicionar GA4 ou Tag Manager somente depois de atualizar consentimento/cookies.
